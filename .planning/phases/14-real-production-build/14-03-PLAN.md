<action>

Create `scripts/build_frontend.ps1` — a PowerShell script that builds the Tauri frontend (Tier 3). This script:
1. Kills any lingering Zedsu processes to prevent file locks
2. Copies static frontend assets (`index.html`, `src/` directory) to `src/ZedsuFrontend-dist/`
3. Runs `cargo build --release` to produce the Tauri binary
4. Copies the binary to `dist/Zedsu/Zedsu.exe`

**Requires: Rust toolchain only. No Node.js/npm.**

```powershell
<#
.SYNOPSIS
    Build the Tauri frontend (Zedsu.exe).
.DESCRIPTION
    Step 1: Kill any lingering Zedsu processes (prevent file locks)
    Step 2: Copy static frontend assets to src/ZedsuFrontend-dist/
    Step 3: Run cargo build --release to produce Zedsu.exe
    Step 4: Copy Zedsu.exe to dist/Zedsu/
.NOTES
    Requires: Rust toolchain (cargo) only. No Node.js/npm.
#>

$ErrorActionPreference = "Stop"
$ProjectRoot = Split-Path -Parent $PSScriptRoot
$FrontendDir = Join-Path $ProjectRoot "src\ZedsuFrontend"
$FrontendDistSrc = Join-Path $ProjectRoot "src\ZedsuFrontend-dist"
$TauriDir = Join-Path $ProjectRoot "dist\Zedsu"

# Step 0: Kill lingering Zedsu processes (prevent file lock errors during rebuild)
$zedsuProcs = Get-Process -Name "Zedsu","ZedsuBackend" -ErrorAction SilentlyContinue
if ($zedsuProcs) {
    Write-Host "[build] Stopping lingering Zedsu processes..."
    $zedsuProcs | Stop-Process -Force -ErrorAction SilentlyContinue
    Start-Sleep -Milliseconds 500
}

# Step 1: Create output directory
New-Item -ItemType Directory -Path $TauriDir -Force | Out-Null

# Step 2: Copy static frontend assets to src/ZedsuFrontend-dist/
Write-Host "[build] Copying static frontend assets to src/ZedsuFrontend-dist/..."
New-Item -ItemType Directory -Path $FrontendDistSrc -Force | Out-Null

# Copy index.html and src/ directory (static HTML/CSS/JS)
$indexSrc = Join-Path $FrontendDir "index.html"
$indexDst = Join-Path $FrontendDistSrc "index.html"
if (Test-Path $indexSrc) {
    Copy-Item $indexSrc $indexDst -Force
    Write-Host "[build] Copied index.html"
} else {
    Write-Host "[build] ERROR: index.html not found at $indexSrc"
    exit 1
}

$srcDir = Join-Path $FrontendDir "src"
$srcDst = Join-Path $FrontendDistSrc "src"
if (Test-Path $srcDir) {
    Copy-Item $srcDir $srcDst -Recurse -Force
    Write-Host "[build] Copied src/ directory"
} else {
    Write-Host "[build] ERROR: src/ directory not found at $srcDir"
    exit 1
}

Write-Host "[build] Frontend assets copied: $FrontendDistSrc"

# Step 3: Build Tauri binary
Write-Host "[build] Building Tauri release binary..."
Push-Location $FrontendDir
cargo build --release
Pop-Location

# Step 4: Copy binary to dist/Zedsu/
$TauriExe = Join-Path $FrontendDir "target\release\zedsu_frontend.exe"
$TauriExeAlt = Join-Path $FrontendDir "target\release\Zedsu.exe"
$DestExe = Join-Path $TauriDir "Zedsu.exe"

if (Test-Path $TauriExe) {
    Copy-Item $TauriExe $DestExe -Force
} elseif (Test-Path $TauriExeAlt) {
    Copy-Item $TauriExeAlt $DestExe -Force
} else {
    Write-Host "[build] ERROR: Tauri binary not found."
    exit 1
}

Write-Host ""
Write-Host "Frontend build complete."
Write-Host "Executable: $DestExe"
```
