# Plan: 14.7-03 — Shared UI Components

**Status:** COMPLETE

## Changes Made

### `src/ZedsuFrontend/src/scripts/ui/normalizers.js` (NEW)

Safe type coercion preventing all crash scenarios:
- `asArray(value)` — returns `[]` for undefined/null/object/string/number
- `asBool(value)` — handles string `"true"`, `"1"`, `"yes"`
- `asNumber(value, fallback)` — parseFloat with fallback
- `normalizeDiscordConfig(discord)` — safe `events` array, `kill_milestones` as number[]
- `normalizeRuntimeConfig(runtime)` — safe field extraction
- `normalizeYoloState(yolo)` — model availability and quality
- `normalizeLogs(logs)` — normalizes string/object log entries

### `src/ZedsuFrontend/src/scripts/ui/dom.js` (NEW)

DOM utilities preventing memory leaks:
- `$()` — cached querySelector
- `clearCache()` — called on page navigation
- `on()` / `off()` — event binding with dedup
- `updateText()` / `updateClass()` — only updates if changed (prevents reflow)

### `src/ZedsuFrontend/src/scripts/ui/page-lifecycle.js` (NEW)

Page lifecycle management:
- `registerPage()` — registers cleanup function
- `unregisterPage()` — calls cleanup + clears DOM cache
- Prevents duplicate listeners, clears stale refs

### `src/ZedsuFrontend/src/scripts/ui/components.js` (NEW)

Reusable component builders:
- `e()` — HTML escape
- `truncate()` — string truncation
- `StatusBadge()` — colored pill
- `Button()` — styled button
- `Toggle()` — toggle switch
- `MetricRow()` — label+value row
- `Card()` — card container
- `Section()` — section with header
- `EmptyState()` — empty list state
- `ErrorState()` — error state with retry
- `Toolbar()` — filter+action row

## Verification
