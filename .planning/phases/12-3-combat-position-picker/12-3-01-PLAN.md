**Class structure:**

```python
"""
Position Picker Overlay — Phase 12.3

Tkinter click-to-capture overlay for picking a single screen position.
Runs in a dedicated non-daemon thread so the HTTP handler can block and wait.
Single-shot: one click closes the overlay immediately.
"""

from __future__ import annotations
import threading
from typing import Optional

try:
    import tkinter as tk
except ImportError:
    tk = None  # type: ignore

from src.utils.windows import get_window_rect


class PositionPickerOverlay:
    """
    Single-shot click-to-capture overlay for picking a screen position.

    Usage (from a dedicated thread):
        overlay = PositionPickerOverlay(window_title="Roblox", position_name="melee")
        overlay.run()          # blocks until overlay closes
        result = overlay.get_result()  # returns result dict or None on timeout

    Result dict shapes:
        {'action': 'confirm', 'x': 0.5, 'y': 0.3, 'name': 'melee'}
        {'action': 'cancel'}
        {'action': 'error', 'message': 'Window not found'}
        {'action': 'error', 'message': 'Click outside game window'}
        None  (timeout)
    """

    def __init__(self, window_title: str, position_name: str) -> None:
        if tk is None:
            raise RuntimeError("tkinter is not available — cannot create overlay")

        self.window_title = window_title
        self.position_name = position_name

        self._win_left: int = 0
        self._win_top: int = 0
        self._win_width: int = 0
        self._win_height: int = 0

        self._tk_root: Optional[tk.Tk] = None
        self._overlay: Optional[tk.Toplevel] = None

        self.result_data: Optional[dict] = None
        self.result_event = threading.Event()
```

**run() method:**
- Create `root = tk.Tk()`, `root.withdraw()`, store in `self._tk_root`
- Call `get_window_rect(self.window_title)` — if None, set `{"action": "error", "message": "Window not found"}`, fire event, clean up root, return
- Compute `_win_width`, `_win_height` from rect
- Create Toplevel overlay, set geometry to `{width}x{height}+{left}+{top}`
- Set overlay attributes: `overrideredirect(True)`, `attributes("-topmost", True)`, `attributes("-alpha", 0.25)`, crosshair cursor, dark background
- Create Canvas filling overlay, bind `<Button-1>` to `_on_click`, bind `<Escape>` to `_cancel()`
- Bind `<Destroy>` to `_on_destroy()` for guard
- `overlay.focus_force()` and `canvas.focus_set()`
- `root.mainloop()`

**Click handler `_on_click(event)`:**
- First: check bounds — if `event.x < 0 or event.x > self._win_width or event.y < 0 or event.y > self._win_height`:
  - Set result: `{"action": "error", "message": "Click outside game window"}`
  - Fire event and call `_close()` immediately (REJECT out-of-bounds, do NOT silently clamp)
- If in-bounds: normalize `norm_x = event.x / self._win_width`, `norm_y = event.y / self._win_height`
- Clamp normalized values: `norm_x = max(0.0, min(1.0, norm_x))`, same for norm_y
- Set result: `{"action": "confirm", "x": norm_x, "y": norm_y, "name": self.position_name}`
- Fire event
- Call `_close()` immediately (single-shot — overlay closes on click, no persistent state)

**_cancel() method:**
- Guard: if event already set, return
- Set `{"action": "cancel"}`, fire event, call `_close()`

**request_cancel(message) method:**
- Guard: if event already set, return
- Set `{"action": "error", "message": message}`, fire event
- If `_tk_root` is not None: `_tk_root.after(0, self._close)`

**_close() method:**
- Destroy Toplevel (if exists) then `root.quit()` + `root.destroy()`

**_on_destroy() method:**
- If result_event NOT set: call `_cancel()` (guards against destroy overwriting confirmed result)

**get_result(timeout=300.0) method:**
- Wait on result_event with timeout, return result_data

**Key differences from RegionSelectorOverlay:**
- Single click captures position (no drag state tracking)
- `_on_click` is the capture — immediate normalize + confirm
- No persistent overlay waiting for multiple captures
- Overlay closes immediately after one click
- Crosshair cursor (same as RegionSelectorOverlay)
