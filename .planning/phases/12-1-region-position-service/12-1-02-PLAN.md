## Task 6: Implement resolve_position()

**read_first:** `src/utils/windows.py` (get_window_rect returns (left, top, right, bottom)), `src/utils/config.py` (lines 749-764, resolve_coordinate pattern for ratio-to-pixel conversion)

**action:**
Implement `resolve_position(config: dict, name: str) -> Optional[Dict[str, Any]]`:
1. Read `config["game_window_title"]` to find window
2. Call `get_window_rect(title)` — D12.1-D02: fresh call every time
3. If rect is None, return None
4. Read `config["combat_positions"][name]` — if not found, return None
5. Convert normalized coords to pixels:
   - `width = right - left`, `height = bottom - top`
   - `abs_x = int(left + pos["x"] * width)`
   - `abs_y = int(top + pos["y"] * height)`
6. Return `{"name": name, "abs_x": abs_x, "abs_y": abs_y, "normalized": {"x": pos["x"], "y": pos["y"]}}`

**acceptance_criteria:**
- `grep -n "def resolve_position" src/services/position_service.py` finds the function
- `grep -n "get_window_rect" src/services/position_service.py` confirms fresh window rect call
- `grep -n "pos\[.x.\] \* width" src/services/position_service.py` confirms normalized-to-pixel conversion
