## Task 2: Implement validate_region()

**read_first:** `src/utils/config.py` (lines 200-220, _normalize_ratio_area pattern)

**action:**
Implement `validate_region(name: str, coords: List[float]) -> tuple[bool, str | None]`:
1. Check `name` is non-empty string (strip whitespace, reject empty)
2. Check `coords` is a list/tuple of length 4
3. Check each value in [0.0, 1.0] — use `0.0 <= v <= 1.0`
4. Check `x1 < x2` and `y1 < y2` (x1 = coords[0], x2 = coords[2], y1 = coords[1], y2 = coords[3])
5. Check area > 0: `(x2 - x1) * (y2 - y1) > 0`
6. Return `(True, None)` on success, `(False, "error message")` on failure

**acceptance_criteria:**
- `grep -n "def validate_region" src/services/region_service.py` finds the function
- `grep -n "x1 < x2" src/services/region_service.py` confirms bbox validation

## Task 6: Implement resolve_region()

**read_first:** `src/utils/windows.py` (get_window_rect returns (left, top, right, bottom)), `src/utils/config.py` (lines 824-844, resolve_outcome_area pattern for ratio-to-pixel conversion)

**action:**
Implement `resolve_region(config: dict, name: str) -> Optional[Dict[str, Any]]`:
1. Read `config["game_window_title"]` to find window
2. Call `get_window_rect(title)` — D12.1-D02: fresh call every time
3. If rect is None, return None
4. Read `config["combat_regions_v2"][name]` — if not found, return None
5. Convert normalized coords [x1,y1,x2,y2] to pixels:
   - `width = right - left`, `height = bottom - top`
   - `abs_x1 = int(left + coords[0] * width)`
   - `abs_y1 = int(top + coords[1] * height)`
   - `abs_x2 = int(left + coords[2] * width)`
   - `abs_y2 = int(top + coords[3] * height)`
6. Return `{"name": name, "abs_area": [abs_x1, abs_y1, abs_x2, abs_y2], "normalized_area": coords}`

**acceptance_criteria:**
- `grep -n "def resolve_region" src/services/region_service.py` finds the function
- `grep -n "get_window_rect" src/services/region_service.py` confirms fresh window rect call
- `grep -n "coords\[0\] \* width" src/services/region_service.py` confirms normalized-to-pixel conversion
