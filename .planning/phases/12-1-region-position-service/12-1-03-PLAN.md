Wire region_service.py and position_service.py to 11 HTTP backend commands. Backend owns persistence (save_config after service returns). Plan 03 also creates src/services/__init__.py and adds get_search_region (which closes Phase 12.0 V6 deferral).
</objective>

<context>
Plans 01 and 02 created service modules that mutate config in-memory but do NOT call save_config(). This plan's backend commands call the service, then persist with save_config(), then reload with load_config() to get the normalized version — same pattern as existing update_config handler. Payload uses `area` (not `coords`) as primary key, with `coords` as optional alias for backwards compatibility. __init__.py is created here so Plans 01/02 can run in parallel without __init__ conflicts.
</context>

## Task 4: Add region commands — get_regions, set_region, delete_region, resolve_region, resolve_all_regions

**read_first:** `src/zedsu_backend.py` (action dispatch chain, lines 709-780), `src/utils/config.py` (save_config pattern at lines 672-675)

**action:**
After the existing YOLO section (after line 778, before the final `else: self._send_json({"status": "error"...})`):

Add all region commands with the pattern:
- Call service function
- On success: `save_config(_app_config)` → `_app_config = load_config()` → send ok response
- On failure: send error with 400/404

```python
            # Phase 12.1: Region commands
            elif action == "get_regions":
                regions = list_regions(_app_config)
                self._send_json({"status": "ok", "regions": regions})

            elif action == "set_region":
                from src.utils.config import save_config, load_config
                payload_data = data.get("payload", {})
                name = payload_data.get("name", "")
                area = payload_data.get("area", payload_data.get("coords", []))
                kind = payload_data.get("kind", "generic")
                threshold = payload_data.get("threshold")
                enabled = payload_data.get("enabled", True)
                label = payload_data.get("label")
                success, error = set_region(_app_config, name, area, kind, threshold, enabled, label)
                if success:
                    save_config(_app_config)
                    _app_config = load_config()
                    self._send_json({"status": "ok"})
                else:
                    self._send_json({"status": "error", "message": error}, 400)

            elif action == "delete_region":
                from src.utils.config import save_config, load_config
                payload_data = data.get("payload", {})
                name = payload_data.get("name", "")
                success, error = delete_region(_app_config, name)
                if success:
                    save_config(_app_config)
                    _app_config = load_config()
                    self._send_json({"status": "ok"})
                else:
                    self._send_json({"status": "error", "message": error}, 404)

            elif action == "resolve_region":
                payload_data = data.get("payload", {})
                name = payload_data.get("name", "")
                resolved = resolve_region(_app_config, name)
                if resolved is None:
                    self._send_json({"status": "error", "message": f"Region not found or window not found: {name}"}, 404)
                else:
                    self._send_json({"status": "ok", "region": resolved})

            elif action == "resolve_all_regions":
                resolved = resolve_all_regions(_app_config)
                self._send_json({"status": "ok", "regions": resolved})
```
