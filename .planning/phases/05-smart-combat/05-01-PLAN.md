<design_rationale>

## Why This Redesign

The original Phase 5 plan had 3 critical flaws:

1. **Frame diff as primary detection** — Too generic for GPO BR with 300ms lag. Would detect zone effects, other players, camera movement as "enemy activity" → massive false positives.

2. **Wrong HP bar region** — Original plan assumed "bottom-left" (enemy HP bar area in generic Roblox). Reality: HP bar appears ABOVE the enemy character's head in GPO BR, near the center of screen (visible in user's image 1 with "HITS:8").

3. **No hit confirmation** — The original plan has no way to know if punches are landing. Red damage numbers + enemy tint (image 3) are the key confirmation signals.

4. **No kill-steal resilience** — With 30-40 players clustering (majority AFK) and 300ms lag, competitors stealing kills is guaranteed. The bot must never stop attacking when enemies are nearby.

5. **Third-person assumption** — Player in third-person makes self-vs-enemy distinction complex. First-person camera simplifies this.

## Key Insights from User Data

- 30-40 players, majority AFK, cluster together
- Server 300ms lag — kill-steal is guaranteed, reactive > predictive
- Enemy HP bar: GREEN, appears above enemy head when close (image 1)
- Red damage numbers fly up when hitting (image 3) — HIT CONFIRM
- Enemy tints RED when taking damage (image 3)
- INCOMBAT timer at top-center (runs continuously during combat)
- Kill icon (skull) at top-right (image 4)
- First-person camera recommended — player always at screen center
- Priority: weak enemies > large clusters > nearest

## Deepwoken Reference Patterns Applied

From ff4500ll/Asphalt-Files-Reuploaded (Deepwoken Fishing Macro 2):
- Pattern: Watch specific color pixels in specific screen region → react immediately
- Speed: Pixel check = 1-5ms vs ImageSearch = 50-200ms
- This is why HSV color detection in Phase 4 was the right foundation

Applied to GPO BR:
- GREEN_HP_BAR: green pixels in configurable near-center region = enemy nearby
- RED_DMG_NUMBERS: red pixels in damage region = confirmed hit
- INCOMBAT_TIMER: white pixels in top-center = combat active
- KILL_ICON: skull shape/color in top-right = kill confirmed

        Uses HSV color region scanning — inspired by Deepwoken Fishing macro approach.
        All regions are configurable and user-picked.

        Detection signals:
        - GREEN_HP_BAR: enemy close enough to show HP bar above head
        - RED_DMG_NUMBERS: damage numbers flying up = confirmed hit
        - PLAYER_HP_BAR: player's own HP bar (for FLEEING trigger)
        - INCOMBAT_TIMER: INCOMBAT indicator active at top-center

        # These are defaults — user adjusts via Settings UI
        HSV_RANGES = {
            "green_hp_bar": {
                # Green HP bar: H=35-85, S=60-255, V=60-255
                "lower": np.array([35, 60, 60], dtype=np.uint8),
                "upper": np.array([85, 255, 255], dtype=np.uint8),
            },
            "red_dmg_numbers": {
                # Red damage numbers: H=0-10 and H=170-180, S=100-255, V=100-255

                "upper2": np.array([180, 255, 255], dtype=np.uint8),
            },
            "player_hp_bar": {
                # Same green HP bar range as enemy (green is green)
                "lower": np.array([35, 60, 60], dtype=np.uint8),
                "upper": np.array([85, 255, 255], dtype=np.uint8),
            },
            "incombat_timer": {
                # White/bright text: H=0-180, S=0-40, V=200-255
                "lower": np.array([0, 0, 200], dtype=np.uint8),
                "upper": np.array([180, 40, 255], dtype=np.uint8),
            },
            "kill_icon": {
                # White skull icon: similar to incombat

                "upper": np.array([180, 30, 255], dtype=np.uint8),

        def _capture_region(self, region):
            """Fast MSS capture of a specific pixel region."""
            if not _MSS_AVAILABLE or np is None:
                return None
            if not region or len(region) != 4:
                return None
            try:
                with mss.mss() as sct:
                    monitor = {
                        "left": int(region[0]),
                        "top": int(region[1]),
                        "width": int(region[2] - region[0]),
                        "height": int(region[3] - region[1]),
                    }
                    screenshot = sct.grab(monitor)
                    img = np.array(screenshot)
                    if img.shape[2] == 4:
                        img = img[:, :, :3][:, :, ::-1]  # BGRA -> RGB
                    return img
            except Exception:
                return None

        def _count_color_pixels(self, img, signal_name):
            """Count pixels matching a signal's color range. Returns (count, total, ratio)."""
            if img is None or img.size == 0:
                return 0, 0, 0.0

            # Convert RGB -> HSV (OpenCV expects BGR)
            hsv = cv2.cvtColor(img, cv2.COLOR_RGB2HSV)
            h, w = hsv.shape[:2]
            total = h * w

            ranges = self.HSV_RANGES.get(signal_name, {})
            if "lower" in ranges and "upper" in ranges:
                # Single range
                lower = ranges["lower"]
                upper = ranges["upper"]
                mask = cv2.inRange(hsv, lower, upper)
            elif "lower1" in ranges:
                # Red hue wrap-around (H=0-10 and H=170-180)
                lower1, upper1 = ranges["lower1"], ranges["upper1"]
                lower2, upper2 = ranges["lower2"], ranges["upper2"]
                mask1 = cv2.inRange(hsv, lower1, upper1)
                mask2 = cv2.inRange(hsv, lower2, upper2)
                mask = cv2.bitwise_or(mask1, mask2)
            else:
                return 0, total, 0.0

            count = cv2.countNonZero(mask)
            return count, total, count / max(1, total)

        def scan_all_signals(self):

            Scan all configured combat signals. Returns dict of signal -> bool.
            Call this once per combat tick (< 20ms total for all 5 regions).
            """
            results = {}

    from enum import Enum, auto
    from time import time

    class CombatState(Enum):
        IDLE = auto()         # Waiting for match to start
        SCANNING = auto()     # Looking for enemies — no signals detected
        APPROACH = auto()     # Enemy detected (far) — moving toward
        ENGAGED = auto()      # Enemy close — attacking
        FLEEING = auto()      # Player HP low — evasive
        SPECTATING = auto()    # Player dead
        POST_MATCH = auto()    # Match ended

    class CombatStateMachine:
        """
        Intelligent combat FSM for GPO BR.

        Transition rules:
        IDLE → SCANNING: match starts (ultimate bar visible)
        SCANNING → APPROACH: enemy detected (green HP bar or frame diff)
        SCANNING → ENGAGED: direct close-range signal
        APPROACH → ENGAGED: enemy close enough (green HP bar detected)
        ENGAGED → FLEEING: player HP drops below threshold
        ENGAGED → SCANNING: enemy lost (no signals for N frames)
        ENGAGED → SPECTATING: death detected
        FLEEING → SCANNING: player HP recovered
        FLEEING → SPECTATING: player died
        Any → SPECTATING: death confirmed
        SPECTATING → POST_MATCH: results screen
        POST_MATCH → IDLE: lobby
        """

        def __init__(self, engine):
            self.engine = engine
            self.state = CombatState.IDLE
            self._state_enter_time = 0
            self._last_enemy_detected = 0
            self._last_incombat_time = 0
            self._kill_count = 0
            self._consecutive_engaged_frames = 0
            self._consecutive_no_signal_frames = 0
            self._scan_direction = 1  # 1 or -1 for camera rotation
            self._scan_timer = 0
            self._cluster_activity_history = []  # track where most activity

            # Thresholds from config
            cfg = engine.app.config.get("combat_settings", {})
            self.disengage_timeout_sec = cfg.get("disengage_timeout_sec", 5.0)
            self.fleeing_hp_threshold = cfg.get("fleeing_hp_threshold", 0.25)
            self.dodge_chance = cfg.get("dodge_chance", 0.12)
            self.camera_scan_interval = cfg.get("camera_scan_interval", 0.5)
            self.kill_steal_resilient = cfg.get("kill_steal_resilient", True)

        @property
        def state_name(self):
            return self.state.name

        def _transition_to(self, new_state):
            old = self.state
            self.state = new_state
            self._state_enter_time = time()
            self.engine.log(f"[COMBAT] {old.name} → {new_state.name}")
            # Reset frame counters on transition
            self._consecutive_engaged_frames = 0
            self._consecutive_no_signal_frames = 0

        def update(self):
            """
            Called every combat tick (~500ms). Returns action dict for engine.
            This is the core decision loop — MUST be fast (< 10ms).
            """
            now = time()

            # Scan all combat signals (fast pixel checks, < 20ms total)
            detector = self.engine._combat_detector
            signals = detector.scan_all_signals()

            enemy_nearby = signals.get("enemy_nearby", False)
            hit_confirmed = signals.get("hit_confirmed", False)
            in_combat = signals.get("in_combat", False)
            player_hp_low = signals.get("player_hp_low", False)
            kill_confirmed = signals.get("kill_confirmed", False)

            # Update kill count
            if kill_confirmed:
                self._kill_count += 1
                self.engine.log(f"[COMBAT] Kill #{self._kill_count} confirmed!")

            # Update engagement history
            self._last_incombat_time = now if in_combat else self._last_incombat_time

            # Frame counter updates
            if enemy_nearby or in_combat or hit_confirmed:
                self._consecutive_engaged_frames += 1
                self._consecutive_no_signal_frames = 0
            else:
                self._consecutive_no_signal_frames += 1
                self._consecutive_engaged_frames = max(0, self._consecutive_engaged_frames - 1)

            # --- STATE TRANSITIONS ---
            target = self._compute_target_state(
                enemy_nearby=enemy_nearby,
                in_combat=in_combat,
                hit_confirmed=hit_confirmed,
                player_hp_low=player_hp_low,
                now=now,
            )
            if target != self.state:
                self._transition_to(target)

            # --- EXECUTE CURRENT STATE ---
            return self._execute_state(signals, now)

        def _compute_target_state(self, enemy_nearby, in_combat, hit_confirmed, player_hp_low, now):
            """Compute the target state based on signals. Override this for custom logic."""
            s = self.state

            # Priority 1: Death
            if self.engine.detect_spectating_state(search_context=self.engine.build_search_context()):
                return CombatState.SPECTATING

            # Priority 2: Fleeing (low HP overrides everything except death)
            if player_hp_low and s not in (CombatState.SPECTATING, CombatState.POST_MATCH, CombatState.IDLE):
                return CombatState.FLEEING

            # Priority 3: Recovery from fleeing
            if s == CombatState.FLEEING and not player_hp_low:
                return CombatState.SCANNING

            # ENGAGED logic: enemy nearby OR incombat OR confirmed hit
            if (enemy_nearby or in_combat or hit_confirmed):
                if s != CombatState.ENGAGED:
                    return CombatState.ENGAGED
                return s  # Stay engaged

            # Disengage: no signals for too long
            no_signal_duration = now - max(self._last_enemy_detected, self._last_incombat_time)
            if no_signal_duration > self.disengage_timeout_sec and s == CombatState.ENGAGED:
                return CombatState.SCANNING

            # SCANNING: default when not engaged
            if s not in (CombatState.IDLE, CombatState.SPECTATING, CombatState.POST_MATCH):
                if not enemy_nearby and not in_combat:
                    return CombatState.SCANNING

            return s

        def _execute_state(self, signals, now):
            """Execute behavior for current state. Returns action dict."""
            s = self.state

            if s == CombatState.SCANNING:
                # Rotate camera + move toward likely player clusters
                self._last_enemy_detected = now
                return {
                    "action": "scan",
                    "scan_direction": self._scan_direction,
